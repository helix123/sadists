## Installation

This package may be installed from CRAN; the latest version may be
found on [github](https://www.github.com/shabbychef/sadists "sadists")
via devtools:


```r
if (require(devtools)) {
    # latest greatest
    install_github(repo = "sadists", username = "shabbychef", 
        ref = "master")
}
```
